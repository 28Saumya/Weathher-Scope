# Weather Scope

Weather Scope is a full-stack weather application built using **Node.js** and **Express**.  
It shows real-time weather information using the **Open-Meteo API** with a modern UI.

---

## Features

- Search weather by city name
- Real-time temperature and wind speed
- Dark / Light mode
- Dynamic UI based on weather
- Live date and time
- Fully responsive design

---

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- Axios
- Open-Meteo API
